## <i class="fas fa-dragon"></i>基本课文

{% note pink "fas fa-dragon" modern %}

甲：わたしわ李です。小野さんですか

乙：はい，そです。小野です

<br/>

甲：もりさんは学生ですか。

乙：いいえ，学生ではありません。会社員です。

<br/>

甲：吉田さんですか。

乙：いいえ，ちかいます。森です。

<br/>

甲：李さはＪＣ企画の社員ですか。

乙：はい，そうです。

{% endnote %}

### 基本课文语法

#### 判断句

1. ---は---です

   == ---是---   。”は”变音为“wa”，放在主语的后面，表示前面的是主语。助词“は”放在句子里面用于提示主题。

2. ---は---ではありません

   == ---不是---  。”では”在口语中有时会发成“じゃ”。

#### 疑问句

1. ---は---ですか。

   == --- 是--- 吗？日语没有问号。。。



## <i class="fas fa-dragon"></i>应用课文

小野：はい，小野です。李秀麗さんですか。

李：はい，李秀麗です。はじめまして。どうもよろしくお願いします。

小野：はじめまして，小野緑です。

森：李さん，こんにちは。

李：吉田さんですか。

森：いいえ，わたしわ吉田じゃありません。森です。

李：あつ，森さんですか。どうもすみません。

森：いいえ。どうぞよろしく。

李：李秀麗です。こちらこそ，よろしくお願いします。

{% endnote %}

单词扩展

{% endnote %}

1. 中国人
   * 中国人；中国の方(敬称)；中国の人
   * ~人  接尾词，不能单独用

2. かぞく（家族）　家人

   * ご家族    别人的家人；你的家人

   * ”ご”　是表示尊敬的接头词，其他接头词还有 (”お”表示尊敬喜好美化；“ご”表示尊敬)

   * お是和语词；ご是汉语词

   * 例：

     * ——ご家族ですか。

       ——家族です。

3. はは（母）　家母（在外人面前称呼自己的妈妈）

   ちち（父）　家父

4. おかあさん（お母さん）　母亲 （对别人说==你妈妈；对妈妈说==妈妈）

   * お　和    さん　都是表示尊敬
   * 如果要表示亲近，お和さん省略一个

5. ~語　　......话（接尾词）

   * ご～　　尊称（接头词）

6. 日语单词分类

   * 体言，整体不能发生变化的单词的统称。有名词，数词，代词，量词
   * 用言，可以发生变化的单词的统称。有动词，形容词，形容动词

7. この；その；あの；どの　｜｜　これ；あれ；それ；どれ

   * この／これ　这，这个/这（量词）离自己近

   * その／それ　那，那个/那（量词）离自己中间距离

   * あの／あれ    那，那个/那（量词）离自己远

   * どの／どれ    哪，哪个/哪（量词）表示疑问

   * ～れ　指事物，是名词。可以单独用

   * ～の　是连接体言的词，是连体词，不能单独用

   * 连体词与体言相连时，翻译的时候中间可以随便加量词。连体词+体言 == 代词

     例：

     * この大学　这所大学
     * この車        这辆车
     * この会社    这所公司

8. なん（何）　（疑问）什么；なに（何）（疑问）什么

   * 一般情况下，读哪个都可以

   * 出现在“が”、”を”前，读“なに”

   * 口语中可以直接读升调，表示疑问

   * “な”可以作接头词

     例：

     * 何曜日
     * 何じ

## <i class="fas fa-dragon"></i>基本课文

{% note pink "fas fa-dragon" modern %}

甲：これはテレビですか。

乙：いいえ，それわテレビではありません。パソコンです。

<br>

甲：それは，何ですか。

乙：かれは日本語の本です。

<br>

甲：森さんのかばんはどれですか。

乙：あのかばんです。

<br>

甲：そのノートは誰のですか。

乙：私のです。

{% endnote %}

### 基本课文语法

1. です
   * 表示判断
   * 表示敬体终止型
2. の
   * 大于等于“的”，表示所属关系或表示时是定语
   * 体言的连体形

## <i class="fas fa-dragon"></i>应用课文

{% note blue "fas fa-dragon" modern %}

小野：李さん，それは何ですか。

李：これですか。家族の写真です。

小野：この方はどなたですか。

李：私のははです。

小野：お母さんはおいくつですか。

李：52（ごじゅに）歳です。

<br>

李：小野さん，これ，どうぞ。

小野：えつ，何ですか。

李：お土産です。

小野：わあ，シルクのハソカチですか。

李：ええ。スワトウのハソカチです。中国の名産品（めいさんひん）です。

小野；どうもあいがとうございます。
